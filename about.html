About This Project

GeoGlance is a web app that uses the REST Countries API to provide detailed information about any country in the world.

It helps users quickly learn about flags, currencies, capitals, languages, populations, and more.

This project was created to demonstrate the power of JavaScript, APIs, and responsive web design.
